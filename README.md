## Project Description

*Beneath the Shattered Sky* is a Unity-based demo created as part of a semester-long Video Game Development course. The concept originates from an original storyline and was adapted into an interactive experience using Unity and C#.

This demo features:
- Short cinematic cutscenes
- Camera transitions
- Interactable objects
- Custom flame textures
- Custom audio tracks

### Objective

The player must:
1. Pick up a torch  
2. Interact with the angel **Amarael**  
3. Light candelabras to reveal pathways  
4. Traverse dangerous terrain suspended over an abyss  

The environment is intentionally dark and atmospheric, encouraging exploration and immersion. As light is introduced into the world, new paths are revealed—guiding the player forward through environmental interaction.

UI & Game Design
This project focuses heavily on player experience, immersion, and environmental storytelling.

Core Design Philosophy
- Minimal instruction, maximum discovery
- Environment-driven progression
- Light as both mechanic and guide
  
Gameplay Flow
- The player begins in a dark environment with limited visibility
- A torch introduces both light and interaction mechanics
- A cutscene introduces Amarael, providing narrative direction
- The player explores and discovers:
- Candles that can be lit
- Light-revealed platforms
- Traversable paths formed dynamically

This creates a repeating gameplay loop:
Explore → Light → Reveal → Progress

Controls
- WASD – Movement
- Mouse – Camera control
- Interact key – Pick up and use objects
- Immersion Features
- Smooth cutscene camera transitions
- Timed music and ambient audio
- Responsive player movement and physics
- Interactive lighting system
- Subtle visual guidance instead of explicit instructions
- Technical Implementation

This project was developed using C# in Unity, with a focus on scalability and clean architecture.

Key Concepts Used
- Object-Oriented Programming (OOP)
- Modular system design
- Player interaction systems
- Inventory handling
- Event-driven interactions
- Physics-based movement
- Lighting systems and effects

Systems were designed to allow future expansion without major rewrites.
